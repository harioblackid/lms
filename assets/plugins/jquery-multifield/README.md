# jQuery Multifield Plugin

A jQuery plugin which allows you to create dynamic field groups.

## USAGE

```<script>
$('#wrapper').multifield({
	section: '.section',
	btnAdd:'.btnAdd',
	btnRemove:'.btnRemove'
});
</script>
```
